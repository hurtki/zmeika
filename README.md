### Linux (x86_64)

```bash
curl -L https://github.com/hurtki/zmeika/releases/latest/download/ascii-snake-linux-amd64 -o ascii-snake && chmod +x ascii-snake && sudo mv ascii-snake /usr/local/bin/
```
